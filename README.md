# Ghana Phone Network Validator
A CLI tool that validates Ghanaian phone numbers and identifies their network provider.

## Features
- Load phone numbers from file
- Validate number format (10 digits, starts with 0)
- Identify network (MTN, Vodafone, AirtelTigo)
- Generate summary report with percentages
- Export report to file
- Filter and view numbers by network

## Networks Supported
- MTN: 024, 054, 055, 059
- Vodafone: 020, 050
- AirtelTigo: 026, 056

## How to Run
python phone_validator.py

- Python 3
- OOP (Classes and Methods)
